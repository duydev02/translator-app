# Keyboard shortcuts

Also viewable in-app via **F1**.

## Translation

| Shortcut | Action |
|----------|--------|
| `Ctrl + Enter` | Translate now |
| `Ctrl + M` | Toggle Table / Inline / Design-Doc mode |
| `Ctrl + Shift + D` | Toggle Forward / Reverse direction |

## Clipboard & files

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + C` | Copy output (excludes the 未定義 section) |
| `Ctrl + S` | Save output to file |
| `Ctrl + R` | Reload `db_schema_output.json` + user map |
| *drag file onto window* | Load file into input (requires `tkinterdnd2`) |

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl + ⌫ (BackSpace)` | Clear input + output |
| `Ctrl + F` | Open search bar |
| `Esc` | Close search bar or popup |

## View

| Shortcut | Action |
|----------|--------|
| `Ctrl + + / =` | Zoom in |
| `Ctrl + -` | Zoom out |
| `Ctrl + 0` | Reset zoom |
| `Ctrl + L` | Toggle line numbers |

## Exclusions dialog

| Shortcut | Action |
|----------|--------|
| *Right-click selected text* | Add / remove exclusion |
| `Ctrl + D` (inside dialog) | Delete selected line(s) |
| `Ctrl + Z` / `Ctrl + Y` | Undo / redo |

## Other

| Shortcut | Action |
|----------|--------|
| `F1` | Show keyboard shortcut overlay |
